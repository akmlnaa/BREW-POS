# BREW POS

BREW POS (Building Reliable E-Web) adalah aplikasi Point of Sale (POS) berbasis web yang dikembangkan menggunakan ASP.NET MVC dan SQL Server untuk membantu operasional bisnis Food & Beverage.

## Fitur

- Login Admin dan Kasir
- User Management
- Menu Management
- Point Of Sale
- Open Bill & Retrieve Bill
- Shift Management
- Payment Processing
- Transaction History
- Product Sales Report
- Export Excel Report

## Teknologi

- ASP.NET MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
- JavaScript
